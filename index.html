About Periodic Table

The periodic table of elements has a total of 118 elements. The 98 elements that occur in nature and the rest are artificially made. The periodic table organizes the elements by their atomic number and chemical properties. The atomic number is the number of protons in an atom. The elements are in rows called periods and columns called groups. Elements in the same group react similarly. The periodic table also shows the state of matter of the elements. Some elements have special names, such as the noble gases or the halogens.
